## **Topic: `isalnum()` Function in C++**

---

### **1. Concept**

`isalnum()` is a **built-in C++ function** (from `<cctype>` header)
that checks whether a given **character is alphanumeric** —
i.e., either **a letter (A–Z, a–z)** or a **digit (0–9)**.

---

### **2. Syntax**

```cpp
bool isalnum(char c);
```

**Parameters:**

* `c` → character to check.
* Passed as `char` or sometimes `int` (ASCII value).

**Return Type:**

* Returns a **non-zero value (true)** if character is alphanumeric.
* Returns **0 (false)** otherwise.

---

### **3. Example**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << isalnum('A') << endl;  // 1 → true
    cout << isalnum('z') << endl;  // 1 → true
    cout << isalnum('9') << endl;  // 1 → true
    cout << isalnum('@') << endl;  // 0 → false
    cout << isalnum(' ') << endl;  // 0 → false
}
```

**Output:**

```
1
1
1
0
0
```

---

### **4. Header File**

```cpp
#include <cctype>   // Required for isalnum, isalpha, isdigit, tolower, etc.
```

> Note: If you include `<bits/stdc++.h>` (like in CP or LeetCode),
> it automatically includes `<cctype>` too.

---

### **5. Common Use Cases**

#### ✅ (a) In palindrome / string cleaning problems:

```cpp
if (isalnum(c))
    clean += tolower(c);
```

#### ✅ (b) To check if user input is only letters or digits:

```cpp
string s = "Code123";
bool allValid = all_of(s.begin(), s.end(), ::isalnum);
```

#### ✅ (c) Tokenizing or ignoring punctuation:

```cpp
if (!isalnum(s[i])) continue; // skip punctuation/symbols
```

---

### **6. Related Functions (also from `<cctype>`)**

| Function     | Checks                         | Example                             |
| ------------ | ------------------------------ | ----------------------------------- |
| `isalpha(c)` | Letter (A–Z, a–z)              | `isalpha('A') → true`               |
| `isdigit(c)` | Digit (0–9)                    | `isdigit('3') → true`               |
| `isalnum(c)` | Letter or Digit                | `isalnum('A'), isalnum('3') → true` |
| `islower(c)` | Lowercase                      | `islower('a') → true`               |
| `isupper(c)` | Uppercase                      | `isupper('Z') → true`               |
| `isspace(c)` | Space / Tab / Newline          | `isspace(' ') → true`               |
| `tolower(c)` | Converts uppercase → lowercase | `'A' → 'a'`                         |
| `toupper(c)` | Converts lowercase → uppercase | `'a' → 'A'`                         |

---

### **7. Why Use `isalnum()` Instead of Manual Check**

Manual check:

```cpp
if ((c >= 'a' && c <= 'z') ||
    (c >= 'A' && c <= 'Z') ||
    (c >= '0' && c <= '9'))
```

Built-in:

```cpp
if (isalnum(c))
```

**Advantage:**

* Cleaner and shorter.
* Handles locale properly (if using non-English chars).
* Less prone to typos.

---

### **8. Notes**

* Works on single characters only.
* Returns an `int`, not strictly `bool`, but you can use it directly in conditions.
* Input must be cast to `unsigned char` if value might exceed ASCII range (important for extended ASCII).

---

---

### **Example inside your palindrome problem**

```cpp
while (left < right && !isalnum(s[left])) left++;
while (left < right && !isalnum(s[right])) right--;
```

→ This simply skips **spaces**, **commas**, **colons**, and **symbols**
so that only letters and numbers are compared.
