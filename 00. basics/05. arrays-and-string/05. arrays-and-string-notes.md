# PART A – ARRAYS

## 1. What is an Array?

Array = Collection of same type elements stored in **continuous memory**.

Example:

```
int arr[5] = {10, 20, 30, 40, 50};
```

Memory view:

```
Index:  0   1   2   3   4
Value: 10  20  30  40  50
```

---

## 2. Why Array is Needed?

Instead of:

```
int a1, a2, a3, a4, a5;
```

We use:

```
int arr[5];
```

Benefits:
• Easy iteration
• Better organization
• Efficient memory handling

---

## 3. Array Declaration

Syntax:

```
dataType arrayName[size];
```

Examples:

```
int arr[5];
float prices[10];
char letters[26];
```

Initialization:

```
int arr[5] = {1,2,3,4,5};
```

---

## 4. Accessing Elements

Using index (zero based):

```
arr[0] = first element
arr[4] = last element
```

Example:

```
cout << arr[2];  // prints 3rd element
```

---

## 5. Iterating an Array

Using for loop:

```cpp
for(int i = 0; i < 5; i++) {
    cout << arr[i] << " ";
}
```

---

## 6. Taking Array Input

```cpp
int arr[5];

for(int i = 0; i < 5; i++) {
    cin >> arr[i];
}
```

---

## 7. Common Mistakes

1. Accessing out of bound index

```
arr[5]   // invalid (last index is 4)
```

2. Not initializing array
3. Confusing index with position

# PART B – STRINGS

## 1. What is a String?

String = sequence of characters.

In C++:

```
string name = "Tanish";
```

---

## 2. Taking String Input

### Method 1 – Single word

```
string name;
cin >> name;
```

Stops at space.

### Method 2 – Full line

```
getline(cin, name);
```

---

## 3. String Internally

```
"Hello"
Indices:
H e l l o
0 1 2 3 4
```

---

## 4. Common String Operations

### Length

```
name.length()
```

### Access character

```
name[0]
```

### Modify

```
name[0] = 'A';
```

---

## 5. Example Program (Array + String)

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5];
    string name;

    cout << "Enter 5 numbers: ";
    for(int i = 0; i < 5; i++)
        cin >> arr[i];

    cout << "Enter your name: ";
    cin >> name;

    cout << "Array: ";
    for(int i = 0; i < 5; i++)
        cout << arr[i] << " ";

    cout << endl;
    cout << "Name: " << name;
    return 0;
}
```

---

## 6. Difference: Array vs String

| Array                  | String                 |
| ---------------------- | ---------------------- |
| Stores numbers/objects | Stores characters      |
| Fixed size             | Flexible (std::string) |
| Uses index             | Uses index             |

---

## 7. Dry Run Example

```
int arr[3] = {5, 10, 15};
cout << arr[1];
```

Output:

```
10
```
